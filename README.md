## Estrutura do projeto

O projeto é composto por três páginas principais: index.html, que reúne a apresentação, seção sobre mim, habilidades e trajetória; projects.html, com os projetos em destaque, e contact.html, com os links e redes sociais. Os estilos ficam em css/style.css, centralizando variáveis, layout, componentes e animações, enquanto js/portfolio.js cuida de toda a interatividade — cursor, partículas, typewriter, scroll reveal e menu mobile.

---

## Critérios atendidos

- **Layout** — CSS Grid e Flexbox em todas as seções
- **Tags semânticas** — `<nav>`, `<header>`, `<main>`, `<footer>`, `<section>`, `<article>`, `<aside>`
- **Navegação** — 3 páginas: `index.html`, `projects.html`, `contact.html`
- **JavaScript** — cursor personalizado, partículas, typewriter, scroll reveal, menu mobile
- **Acessibilidade** — `aria-label`, `aria-live`, foco visível, `prefers-reduced-motion`
- **README** — este arquivo, atualizado em múltiplos commits

---

## Funcionalidades JavaScript

- **Cursor personalizado** — ponto + anel animados que seguem o mouse
- **Partículas interativas** — rede de pontos animada no hero via Canvas API
- **Efeito typewriter** — texto que alterna com animação de digitação
- **Scroll reveal** — elementos aparecem suavemente ao rolar a página
- **Menu mobile** — hamburger menu responsivo com animação
